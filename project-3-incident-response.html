인프라 보안 강화 (3대 개선)

  • Bastion Host 도입 — 업무망 PC가 DC 서버와 직접 통신하던 구조를 제거하고, 관리자만 Bastion(점프 호스트)을 경유해 RDP(3389)/SSH(22)로만 접근하도록 재설계
  • 로그 수집 단방향 통제 — 모든 영역이 SIEM 서버에 직접 연결되던 구조를 개선, 업무망·서버망 각각에 Log Forwarder를 배치하고 Forwarder → SIEM 단방향(Outbound)만 허용
  • Log Forwarder 경유 구조 — DEV_PC → SIEM 직접 연결 경로를 완전히 제거해 업무망 침해 시에도 보안관제망을 독립적으로 보호
